Welcome to splat2api

This API shows information about map schedules in Splatoon 2.
By Terax235 - Source Code

Documentation

Endpoint Method Description
/schedules GET Returns the next 12 map schedules
/schedules/current GET Returns the current map schedule
/coop_schedules GET Returns the next coop mode schedules
/coop_schedules/current GET Returns the current coop mode schedules (if there is no coop currently, it'll return an object containing { "active" : false }
/next_update GET Returns the time until the APIs information gets updated (e.g. until the next map rotation) in hours, minutes and seconds.
